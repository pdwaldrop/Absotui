# Contributing to Absotui

Thanks for your interest in **Absotui**! 🦜

## ⚠️ Beta Version
Absotui is a fork of [Toutui](https://github.com/AlbanDAVID/Toutui), which was archived by its original author in December 2025. This fork exists to keep the project alive and as a way to learn Rust, so the code and workflow here are still evolving. There are known bugs (check [known_bugs](known_bugs.md) and open issues).

## 🔁 Branching workflow 
This project follows this [branching workflow](https://gist.github.com/digitaljhelms/4287848). 

## 💬 How to Contribute
- **Share your theme**: Check [here](https://github.com/AlbanDAVID/Toutui-theme) (the original project's theme repo — themes there should still be compatible).
- **Suggestions/feedback**: Open an issue (feature request) or use [discussions](https://github.com/pdwaldrop/Absotui/discussions).
- **Bugs**: Report bugs not listed in issues or [known bugs](known_bugs.md). Use the appropriate issue section (Installation issue or bug report).
- **Code**: Fork the repo, create a branch, and submit a pull request.
