# Contributing to Absotui

Thanks for your interest in **Absotui**! 🦜

## ⚠️ Beta Version
Absotui is still in heavy development, so the code and workflow here are still evolving. There are known bugs (check [known_bugs](known_bugs.md) and [open issues](https://github.com/pdwaldrop/Absotui/issues)).

## 🔁 Branching workflow 
This project follows this [branching workflow](https://gist.github.com/digitaljhelms/4287848). 

## 💬 How to Contribute
- **Share your theme**: Check [here](https://github.com/AlbanDAVID/Toutui-theme) (Toutui's theme repo — themes there should still be compatible).
- **Suggestions/feedback**: Open a [Feature Request issue](https://github.com/pdwaldrop/Absotui/issues/new/choose) or start a [discussion](https://github.com/pdwaldrop/Absotui/discussions).
- **Bugs**: Report bugs not already listed in [issues](https://github.com/pdwaldrop/Absotui/issues) or [known bugs](known_bugs.md), using the Bug Report issue template. Installation/update/uninstall problems specifically have their own template.
- **Code**: Fork the repo, create a branch, and submit a pull request. Run `cargo build`, `cargo clippy --all-targets`, and `cargo test` before opening it — see [CLAUDE.md](CLAUDE.md) for the full command list and codebase architecture notes.
